# 复杂岩性边坡动力响应与稳定性评价系统

> 基于 Vue 3 + ECharts 的专业岩土工程数值分析与综合评估平台，适用于边坡防灾减灾领域的科研与工程实践。

## 📋 项目简介

本系统面向**复杂岩性边坡**在地震等动力作用下的响应分析与稳定性评价需求，提供从数据管理、动力响应分析、能量演化研究到多方法综合稳定性评价的完整工作流，并支持实时监测预警与报告导出功能。

## ✨ 功能模块

### 1. 首页总览

- 系统运行状态实时监控（CPU、内存、磁盘、温度）
- 8 个传感器在线状态与电量监测
- 能量演化趋势图表与稳定性等级分布（柱状图/饼图切换）
- 三级预警信息展示（红/橙/蓝）
- 最近评价结果与操作时间线

### 2. 基础数据管理

- **边坡信息**：岩性类型、坡高、坡角、内聚力、内摩擦角、容重等 13 个参数的增删改查
- **工况信息**：地震波类型选择与波形参数配置
- **监测点信息**：传感器部署位置与坐标配置
- 支持搜索、筛选、分页功能

### 3. 动力响应分析

- **参数配置**：地震波参数（PGA、持续时间、时间步长、阻尼比）、岩体力学参数、分析设置
- **六维分析**：
  - 时程曲线（加速度/速度/位移）
  - FFT 频谱分析与功率谱密度
  - 反应谱（Sa / Sv / Sd，5% 阻尼）
  - Arias 强度
  - 速度-位移相平面图
  - 多监测点响应对比

### 4. 能量演化分析

- 支持输入能、耗散能、弹性能、动能的时间序列分析
- 标准 / 分阶段 / 逐层三种分析模式
- 能量分布饼图、能量平衡 Sankey 图、能量速率曲线
- 能量异常自动识别与突变检测

### 5. 稳定性评价

提供三种国际通用评价方法：

| 方法 | 原理 |
|------|------|
| **加权综合评分法** | 基于 AHP 层次分析法确定权重，6 项指标加权评分 |
| **模糊综合评判法** | 隶属度函数 → 模糊关系矩阵 → 综合评判 |
| **灰色关联分析法** | 参考序列与比较序列关联度排序 |

**6 项评价指标**：最大位移、峰值加速度、最大应力、能量异常系数、耗散比、放大系数

**稳定性等级体系**：

| 等级 | 得分范围 | 风险级别 |
|------|---------|---------|
| 稳定 | ≥ 85 | 低风险 |
| 基本稳定 | 70 ~ 84 | 中低风险 |
| 欠稳定 | 55 ~ 69 | 中高风险 |
| 不稳定 | 40 ~ 54 | 高风险 |
| 极不稳定 | < 40 | 极高风险 |

附加功能：雷达图可视化、敏感性分析（龙卷风图）、安全系数计算、三种方法结果对比、风险矩阵

### 6. 实时监测

- 加速度、位移、耗散能、安全评级实时数据卡片
- 4 通道同步波形显示与实时 FFT 频谱
- 8 个传感器状态管理与自检
- 四级预警机制（蓝/黄/橙/红）
- 预警记录与系统日志

### 7. 结果导出

- 多章节报告预览与打印
- 多工况数据对标分析

### 8. 系统设置

- 18 项分析参数配置
- AHP 权重矩阵与一致性检验阈值
- 5 级稳定性评价分数阈值
- 预警参数与通知方式配置（邮件/短信/声音）
- 显示偏好与权限管理

## 🔬 核心算法

### AHP 层次分析法

列归一化 → 行平均 → 权重向量，自动计算 $\lambda_{max}$、CI、CR 一致性指标（CR < 0.1 视为通过）。

### 安全系数计算

$$F_s = \frac{c \cdot h / \sin\alpha + (W \cos\alpha - u) \tan\varphi}{W \sin\alpha + u \cos\alpha}$$

动力安全系数引入水平地震系数 $k_h = \text{PGA} \times \text{放大系数} / g$，采用 Bishop 简化圆弧滑动法。

### 地震波生成

支持 4 种波型：**汶川波**（2008 年实录）、**El Centro 波**（1940 年标准波）、**人工波**（规范设计波）、**正弦波**。多频率叠加 + 指数衰减包络函数。

### 反应谱计算

周期范围 0.02s ~ 6.0s，采用 Newmark 数值积分法计算谱加速度（Sa）、谱速度（Sv）、谱位移（Sd）。

### Arias 强度

$$I_a = \frac{\pi}{2g} \int_0^T a^2(t) \, dt$$
